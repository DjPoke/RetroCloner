# Retro Cloner

## What is Retro Cloner ?

Retro Cloner is a tool made with lua/löve2d to create retro games for Windows, Linux and Mac easily, imitating Amstrad CPC, C64 or ZX Spectrum graphics.

This project is in Work In Progress state. For now, it is developped under Linux Mint, but It should works both on Windows or MacOS.

![screenshot1](screenshot1.jpg)

## Short tutorial
0) Install Löve2d 11.5, go inside Retro Cloner folder, open a terminal here and type "love ."
1) Create a new project. Type a filename and choose the computer graphics preset you want.
2) Don't forget to save your project frequently.
3) Edit your palette, if possible! At the bottom of the window, you can see the keys to edit the palette.
4) Edit the parameters of the game screen.
5) Edit blocks. Blocks are ground, walls, stairways and other stuffs.
6) Edit sprites images.
7) Create animations, choosing a few sprites images by animation.
8) Create your actors. You can choose what kind of game you'll play, by choosing it on actor 1. Other actors can be enemies or bonus/malus.
9) Draw your levels. You can draw blocks, or position actors instances.
10) Edit game data.
11) Import wav sounds. [WIP]
12) Import ogg musics.
13) Import images (intro, interface, etc.). They will be converted to the graphic preset choosen at step 1.
14) You can test your game when you want to. [WIP]
15) You can export your game when it is finished.
